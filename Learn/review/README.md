## Review 模板

每次 review 建议包含以下结构：

1. 整体架构（调用链路图）
2. 各层分析（按模块逐个过）
3. 发现的问题（按优先级 P1/P2/P3 排列）
4. 评分（正确性/可靠性/性能/安全性/可观测性/可维护性）
5. 优先修复建议
