为什么不先重启

重启只能消除表象，真正影响恢复时间的通常是下面 5 件事：

  • 错误率是否同步上升
  • 队列堆积是否扩大
  • Redis / DB timeout 是否集中出现
  • 最近发布是否改变调用路径
  • 下游重试是否放大流量

一个比较稳的节奏

先确认影响面，再拉取 metrics 和 error logs，最后把历史相似案例作为辅助证据，不把历史标签当实时事实。
